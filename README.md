### Лабораторная работа №1: Прямоугольники
**Задание:** Определить площадь части прямоугольника, расположенной в I координатной четверти.

**Файлы:**
- `headers.hpp`: описание структур `Point` и `Rectangle`.
- `Rectangle.cpp`: логика расчета площади.
- `main.cpp`: точка входа, инициализация данных и вывод.
